
# Star Wars Memory Card Game

This application was built with React which that allows users to play a memory card game.

[Live Demo]()

## Installation

Install my-project with npm

```bash
  git clone https://github.com/asadjawaid/Memory-Card-Game.git
  cd Memory-Card-Game
  npm install
  npm start
```
